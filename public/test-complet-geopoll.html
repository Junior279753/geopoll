Test Complet GeoPoll

Guide de test exhaustif de l'application du début à la fin

1 Préparation et Vérification du Serveur

  • Démarrer le serveur
    Exécuter npm run dev dans le terminal
  • Vérifier l'accès
    Ouvrir http://localhost:3001
  • Vérifier la base de données
    S'assurer que database/geopoll.db existe

2 Interface Publique et Inscription

  • Page d'accueil
    Vérifier le design, les liens, les animations
  • Inscription utilisateur
    Créer un nouveau compte avec des données valides
  • Vérifier l'email de confirmation
    Simuler la vérification email (si implémentée)
  • Connexion utilisateur
    Se connecter avec le compte créé
Page d'accueil Inscription Connexion

3 Dashboard Utilisateur

  • Accès au dashboard
    Vérifier la redirection après connexion
  • Informations utilisateur
    Vérifier l'affichage du profil et des statistiques
  • Statut d'approbation
    Vérifier le message "En attente d'approbation"
  • Modification du profil
    Tester la mise à jour des informations personnelles
Dashboard Profil

4 Interface d'Administration

Identifiants Admin

Email : admin@geopoll.ci
Mot de passe : admin123
  • Connexion admin
    Se connecter avec les identifiants admin
  • Dashboard admin
    Vérifier les statistiques et la carte des demandes d'approbation
  • Gestion des utilisateurs
    Approuver l'utilisateur créé en phase 2
  • Système de notifications
    Tester les notifications (clic sur l'icône cloche)
  • Section Paramètres
    Vérifier l'accès et le chargement des paramètres
Interface Admin Test Auth Admin Test Notifications

5 Test du Cycle Complet

  • Approuver l'utilisateur
    Depuis l'interface admin, approuver l'utilisateur créé
  • Retour au dashboard utilisateur
    Vérifier que le statut a changé (actualiser la page)
  • Accès aux sondages
    Vérifier que l'utilisateur peut maintenant accéder aux sondages
  • Système de récompenses
    Tester les gains et demandes de retrait

Checklist de Validation Finale

  • L'inscription utilisateur fonctionne
  • La connexion utilisateur fonctionne
  • Le dashboard utilisateur s'affiche correctement
  • L'interface admin est accessible
  • Les notifications admin fonctionnent
  • La gestion des utilisateurs fonctionne
  • L'approbation des utilisateurs fonctionne
  • La section Paramètres est accessible
  • L'actualisation automatique fonctionne
  • Le cycle complet utilisateur → admin → approbation → accès fonctionne

Points d'Attention

  • Base de données : Vérifiez que la BDD contient des données de test
  • Permissions : Certaines fonctions nécessitent une approbation admin
  • Tokens : Les sessions peuvent expirer, reconnectez-vous si nécessaire
  • Console : Surveillez les logs pour identifier les erreurs
